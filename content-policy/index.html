記事制作ポリシー

「転職エージェント比較ラボ」では、転職を検討する皆さまに信頼性の高い情報をお届けするため、独自の評価基準と厳格な制作プロセスに基づいて記事を制作しています。本ポリシーでは、当サイトの記事制作における方針と基準を公開します。

1. 基本方針

当サイトは、転職エージェントの比較・レビュー情報を提供するメディアサイトとして、以下の基本方針に基づき運営しています。

  1. ユーザーファースト：読者の転職成功に役立つ情報提供を最優先とします。
  2. 客観性と公平性：特定のエージェントに偏らない、客観的な比較情報を提供します。
  3. 透明性：評価基準、情報源、アフィリエイト関係を明確に開示します。
  4. 正確性：公式情報と独自調査に基づき、正確な情報の掲載に努めます。
  5. 最新性：定期的に情報を更新し、最新の状況を反映した記事を提供します。

2. 転職エージェントの評価基準

当サイトでは、以下の6つの評価軸に基づき、各転職エージェントを総合的に評価しています。

2-1. 求人数・求人の質

各エージェントが保有する公開求人数・非公開求人数を調査し、求人の量を定量的に評価します。また、求人の質についても、業界別・職種別の充実度、年収帯のバリエーション、大手企業・優良企業の求人割合などを総合的に判断します。

2-2. サポート体制（面接対策、書類添削等）

キャリアカウンセリング、履歴書・職務経歴書の添削、面接対策、年収交渉の代行など、転職活動における各種サポートの充実度を評価します。オンライン面談の対応可否や、サポート提供の時間帯なども確認しています。

2-3. 担当者の質・対応力

キャリアアドバイザーの専門知識、業界理解度、コミュニケーション力、レスポンスの速さなど、担当者の質と対応力を評価します。担当者変更の対応の柔軟さも評価項目に含まれます。

2-4. 利用者の口コミ・評判

実際にサービスを利用した方の口コミ・評判を調査・分析します。SNS、口コミサイト、アンケート調査など複数のソースから情報を収集し、良い評価・悪い評価の両方を公平に紹介します。

2-5. 対応業界・職種の幅

IT・Web、メーカー、金融、コンサル、医療など、対応する業界の幅広さを評価します。また、営業、エンジニア、管理部門、経営層など、対応する職種のカバー範囲も確認しています。総合型か特化型かの特性も明記します。

2-6. 利用のしやすさ（登録～内定までの流れ）

登録のしやすさ（所要時間、必要情報）、初回面談までのスピード、求人紹介のタイミング、退会手続きの簡便さなど、利用全体を通したユーザー体験を評価します。Web・アプリの使いやすさも評価対象です。

3. 記事制作フロー

当サイトでは、以下の5ステップで記事を制作しています。

ステップ1：調査

  • 各転職エージェントの公式サイトから最新の情報を収集
  • 利用者の口コミ・評判を複数のプラットフォームから調査
  • 業界動向や転職市場のトレンドを分析
  • 競合サイトの情報と照らし合わせて網羅性を担保

ステップ2：執筆

  • 調査結果に基づき、当サイトの評価基準に沿って記事を構成
  • 読者にとって分かりやすい表現を心がけ、専門用語には適宜解説を付加
  • メリットだけでなくデメリットも正直に記載
  • 具体的な数値やデータを用いて客観性を担保

ステップ3：レビュー

  • 執筆者とは別の編集者が記事の内容を確認
  • 情報の正確性、表現の適切さ、読みやすさをチェック
  • 各エージェントの公式情報と照合し、事実確認を実施
  • 景品表示法等の法令に抵触する表現がないか確認

ステップ4：公開

  • レビューを通過した記事のみを公開
  • 公開日を明記し、情報の鮮度を読者に提示

ステップ5：定期更新

  • 公開済み記事は定期的（原則3ヶ月ごと）に内容を見直し
  • 各エージェントのサービス変更、求人数の増減、料金改定等を反映
  • 更新日を記事に明記し、最新性を担保
  • 読者からのフィードバックも記事改善に活用

4. 情報の正確性への取り組み

当サイトでは、掲載情報の正確性を確保するため、以下の取り組みを行っています。

  1. 一次情報の重視：各転職エージェントの公式サイト、プレスリリース、IR情報などの一次情報を優先的に参照します。
  2. 複数ソースでの確認：重要な情報は複数の情報源で裏付けを取り、信頼性を高めています。
  3. 情報鮮度の管理：掲載情報の確認日・更新日を管理し、古い情報が放置されないよう定期的にチェックしています。
  4. 誤情報の訂正：誤りが判明した場合は速やかに訂正し、必要に応じて訂正箇所を明示します。
  5. 読者からの指摘対応：読者からの情報修正のご指摘には迅速に対応し、正確な情報への更新を行います。

5. ランキング・おすすめの方針

  1. ランキングは、前述の6つの評価基準に基づく総合評価で順位を決定しています。
  2. アフィリエイト報酬の有無や金額は、ランキングの順位に影響しません。
  3. ランキングの根拠となる評価ポイントは、各記事内で明示しています。
  4. おすすめエージェントの選定は、ユーザーの目的（年代、業界、キャリアレベル等）に応じて最適な提案を行います。

6. アフィリエイト広告について

  1. 当サイトは、各種アフィリエイトプログラムに参加しており、サイト内のリンクを通じてサービスへの申し込みがあった場合、当社が報酬を受け取ることがあります。
  2. アフィリエイト報酬は、当サイトの運営費用（サーバー費用、調査費用、人件費等）に充当しています。
  3. アフィリエイト関係の有無に関わらず、すべてのエージェントを同一の評価基準で公平に評価します。
  4. 広告主からの依頼により、事実と異なる情報を掲載することはありません。

7. 景品表示法への対応

当サイトは、不当景品類及び不当表示防止法（景品表示法）を遵守し、以下の点に留意して記事を制作しています。

  1. 優良誤認表示（実際より著しく優良であると示す表示）を行いません。
  2. 有利誤認表示（取引条件が実際より著しく有利であると示す表示）を行いません。
  3. 広告であることを隠したステルスマーケティングは行いません。
  4. アフィリエイトリンクを含む記事では、その旨を明示します。
